# devkit

Curated dev cheatsheets, templates, and concept notes I keep handy while working.

Only the generic, reusable pieces — no machine- or project-specific config.

## cheatsheets/

Quick references for git, docker, tmux, fzf, jq, ripgrep, curl, ssh, nginx,
kubectl, sed/awk, regex, systemd, PowerShell, and more.
See [cheatsheets/README.md](cheatsheets/README.md) for the full index.

## templates/

Boilerplate starting points — Docker Compose (Spring + Postgres), a Makefile
template, and small `pg-dump` / `port` helper scripts.

## notes/

Short concept notes — CAP theorem, RSA/AES hybrid encryption, SQL injection
classes, SSH vs SSL, virtualization, Neovim internals, and Postgres / Oracle /
Vertica operational tips.
